Use Case:

Вариант использования - Осуществить поиск вакансии

Область действия - Мобильное приложение и Вебсайт портала по поиску работы

Уровень - Цели пользователя-соискателя

Основное действующее лицо - Пользователь-соискатель

Гарантии успеха - Поиск вакансий функционирует

Триггер - В любой момент, когда пользователь находится на странице поиска и нажимает кнопку "Искать"

Описание: 1. Пользователь заходит на страницу поиска. 2. Нажимает "Искать". 3. Система осуществляет поиск вакансий, соответствующих критериям.

Расширения: 1. Пользователь получает информацию о недоступности сервиса. 2. Пользователь получает инормацию о том, что ни найдено ни одной вакансии
